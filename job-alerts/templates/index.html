Setup Guide

This app runs automatically every morning via GitHub Actions. You need to add the following secrets to your GitHub repository (Settings → Secrets and variables → Actions):

  1. ADZUNA_APP_ID and ADZUNA_APP_KEY — free API keys from developer.adzuna.com (register, create an app, copy ID + Key)
  2. REED_API_KEY — free key from reed.co.uk/developers/jobseeker (optional but recommended for broader UK coverage)
  3. GMAIL_USER — your Gmail address: evanelder1992@gmail.com
  4. GMAIL_APP_PASSWORD — a Gmail App Password (Google Account → Security → 2-Step Verification → App passwords; create one for "Mail")

Once secrets are set, the GitHub Actions workflow will run at 07:00 UTC daily and email results to evanelder1992@gmail.com.

You can also trigger it manually from the Actions tab in GitHub (look for the Daily Job Alert Email workflow → Run workflow).

Running locally:

cd job-alerts
pip install -r requirements.txt
export ADZUNA_APP_ID=xxx  ADZUNA_APP_KEY=yyy
export GMAIL_USER=evanelder1992@gmail.com  GMAIL_APP_PASSWORD=zzz
python job_alerts.py            # live send
python job_alerts.py --dry-run  # skip actual email
